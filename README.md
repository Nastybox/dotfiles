# Dotfiles
Arch Linux, KDE Plasma, Zsh and Neovim.
Updating Neovim config.
Soon i3 config.

## Installation
```
cd $HOME
git clone https://github.com/alessiocelentano/dotfiles
mv dotfiles/.* ./
pacman -S --needed - < pkg/pkglist.txt
paru -S --needed - < pkg/pkglist-aur.txt
```
